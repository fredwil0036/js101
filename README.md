🚀 50-Day JavaScript Challenge (with Description)
🟢 Beginner (Day 1–10)

Day 1 – Hello World
Write a JavaScript program that prints "Hello World" in the console.

Day 2 – Sum of Two Numbers
Create a function that takes two numbers and returns their sum.

Day 3 – Even or Odd Checker
Write a program that checks if a number is even or odd.

Day 4 – Reverse a String
Create a function that reverses a string.
Example: "hello" → "olleh"

Day 5 – Palindrome Checker
Check if a word reads the same forward and backward.
Example: "racecar" → palindrome.

Day 6 – Find Largest Number in Array
Given an array of numbers, return the largest value.

Day 7 – Count Vowels
Count how many vowels (a, e, i, o, u) are in a string.

Day 8 – Factorial Calculator
Calculate the factorial of a number.
Example: 5! = 120.

Day 9 – FizzBuzz
Print numbers from 1–100:

divisible by 3 → "Fizz"

divisible by 5 → "Buzz"

divisible by both → "FizzBuzz"

Day 10 – Random Number Generator
Generate a random number between 1 and 100.

🟡 Beginner–Intermediate (Day 11–20)

Day 11 – Remove Duplicates from Array
Create a function that removes duplicate values from an array.

Day 12 – Find Second Largest Number
Return the second largest number in an array.

Day 13 – Capitalize Words
Capitalize the first letter of each word in a sentence.

Day 14 – Sort Array
Sort an array of numbers in ascending order.

Day 15 – Count Word Frequency
Count how many times each word appears in a sentence.

Day 16 – Flatten Nested Array
Convert a nested array into a single array.

Day 17 – Find Missing Number
Given a sequence of numbers, find the missing number.

Day 18 – Group Objects by Property
Group objects in an array based on a property value.

Day 19 – Remove Falsy Values
Remove values like false, 0, "", null, undefined, NaN from an array.

Day 20 – Chunk Array
Split an array into smaller arrays of a specified size.

🟠 Intermediate (Day 21–35)

Day 21 – Fibonacci Sequence
Generate Fibonacci numbers up to n.

Day 22 – Prime Number Checker
Check if a number is prime.

Day 23 – Anagram Checker
Check if two strings are anagrams.
Example: "listen" and "silent".

Day 24 – Longest Word in Sentence
Find the longest word in a sentence.

Day 25 – Title Case Converter
Convert a sentence so each word starts with a capital letter.

Day 26 – Array Intersection
Find common elements between two arrays.

Day 27 – Unique Values Between Arrays
Find elements that appear in one array but not the other.

Day 28 – Rotate Array
Rotate array elements left or right.

Day 29 – Debounce Function
Create a function that delays execution until a user stops triggering it.

Day 30 – Throttle Function
Limit how often a function can execute.

Day 31 – Deep Clone Object
Create a deep copy of an object.

Day 32 – Implement map()
Create your own version of Array.map().

Day 33 – Implement filter()
Create your own version of Array.filter().

Day 34 – Implement reduce()
Create your own version of Array.reduce().

Day 35 – Event Emitter
Build a small event system where functions can subscribe and trigger events.

🔴 Advanced / Backend-Style (Day 36–50)

Day 36 – Promise Example
Create and use a Promise to simulate an asynchronous task.

Day 37 – Fetch API Data
Fetch data from a public API using async/await.

Day 38 – Retry Failed API Request
Retry an API call if it fails.

Day 39 – Rate Limiter
Limit how many times a function can be called per second.

Day 40 – Cache API Response
Store API results to avoid repeated calls.

Day 41 – Logger System
Build a simple logger that logs messages with timestamps.

Day 42 – UUID Generator
Generate unique IDs similar to UUID.

Day 43 – Queue Data Structure
Implement a queue with enqueue and dequeue.

Day 44 – Stack Data Structure
Implement a stack with push and pop.

Day 45 – LRU Cache
Create a Least Recently Used cache system.

Day 46 – Simple Router
Create a simple request router like in Express.

Day 47 – File Read/Write (Node.js)
Read and write files using the Node.js filesystem module.

Day 48 – Simple REST API
Create a small API with routes like:

GET /users

POST /users

Day 49 – Authentication Logic
Simulate login authentication using tokens.

Day 50 – Todo API Project
Build a mini REST API for todos:

create todo

read todos

update todo

delete todo
